Bold360 AI Web API and SDK Demo

This demo shows how you can run the Bold360 AI Web API and SDK in your own environment.

Setup: Before running the demo, you need to edit conf.js for your environment and place the files in your web server.

Web API

  • Embedded Search Widget - embeddedWidget
  • Embedded Support Center Widget - supportCenter
  • Floating Widget - floatingWidget
  • Conversational API
